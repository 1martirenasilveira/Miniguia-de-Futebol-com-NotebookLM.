# 📒 Miniguia de Estudo
  # Resumos Estruturados
- O futebol moderno surgiu na Inglaterra no século XIX.  
- A FIFA organiza a Copa do Mundo desde 1930.  
- Evolução tática: do 2-3-5 inicial até o 4-3-3 e 4-2-3-1 atuais.  
- Impacto cultural: paixão nacional em diversos países, influência econômica e social.

# Glossário
- **Gol**: quando a bola cruza totalmente a linha entre as traves.  
- **Impedimento**: infração quando o jogador recebe a bola em posição irregular.  
- **Tática**: organização dos jogadores em campo para otimizar desempenho.  
- **Copa do Mundo**: maior torneio internacional de seleções, realizado a cada 4 anos.
